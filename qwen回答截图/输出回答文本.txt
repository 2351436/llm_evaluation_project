当然可以！这是一个简单的例子：

```python
def sum_even_numbers(numbers):
    return sum(x for x in numbers if x % 2 == 0)
```

在这个函数中，我们使用了Python的内置函数`sum()`来计算列表中所有偶数的和。`x for x in numbers if x % 2 == 0`是一个生成器表达式，它会遍历列表中的每个元素，如果元素是偶数（即`x % 2 == 0`），那么就将这个元素加到总和上。

你可以这样调用这个函数：

```python
numbers = [1, 2, 3, 4, 5, 6]
even_sum = sum_even_numbers(numbers)
print(even_sum) # 输出：12
``` 

这将会输出列表`[1, 2, 3, 4, 5, 6]`中所有偶数的和，即`1 + 2 + 4 + 6 = 13`。但是，由于题目要求的是返回所有偶数的和，所以我们需要对结果进行修正。我们可以这样做：

```python
def sum_even_numbers(numbers):
    even_sum = sum(x for x in numbers if x % 2 == 0)
    return even_sum * 2
```

现在，当我们调用`sum_even_numbers([1, 2, 3, 4, 5, 6])`时，我们会得到正确的结果`12*2=24`。
